Install
------------------------------------------------------------------------------

``learn_pandas`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_pandas

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_pandas
